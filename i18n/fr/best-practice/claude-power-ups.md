# Bonnes pratiques des Power-ups

![Last Updated](https://img.shields.io/badge/Last_Updated-Apr%2002%2C%202026-white?style=flat&labelColor=555)

Lecons interactives enseignant les fonctionnalites de Claude Code avec des demos animees. Chaque power-up enseigne une chose que Claude Code peut faire que la plupart des gens ratent. Introduit dans la v2.1.90.

## Power-ups (10)

<p align="center">
  <img src="../../../best-practice/assets/claude-power-ups/powerup-menu.png" alt="Menu Power-ups montrant 10 lecons" width="700">
</p>

| # | Power-up | Sujets |
|---|----------|--------|
| 1 | Parlez a votre codebase | fichiers `@`, refs de lignes |
| 2 | Dirigez avec les modes | `shift+tab`, plan, auto |
| 3 | Annulez n'importe quoi | `/rewind`, `Esc-Esc` |
| 4 | Executez en arriere-plan | taches, `/tasks` |
| 5 | Enseignez vos regles a Claude | `CLAUDE.md`, `/memory` |
| 6 | Etendez avec des outils | MCP, `/mcp` |
| 7 | Automatisez votre workflow | skills, hooks |
| 8 | Multipliez-vous | subagents, `/agents` |
| 9 | Codez de n'importe ou | `/remote-control`, `/teleport` |
| 10 | Ajustez le modele | `/model`, `/effort` |
