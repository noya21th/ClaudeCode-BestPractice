# أفضل ممارسات Power-ups

![Last Updated](https://img.shields.io/badge/Last_Updated-Apr%2002%2C%202026-white?style=flat&labelColor=555)

دروس تفاعلية تعلّم ميزات Claude Code مع عروض متحركة. كل power-up يُعلّم شيئاً واحداً يمكن لـ Claude Code فعله ويغفل عنه معظم الناس. قُدّم في الإصدار v2.1.90.

## Power-ups (10)

<p align="center">
  <img src="../../../best-practice/assets/claude-power-ups/powerup-menu.png" alt="قائمة Power-ups تعرض 10 دروس" width="700">
</p>

| # | Power-up | المواضيع |
|---|----------|----------|
| 1 | تحدث مع قاعدة كودك | ملفات `@`، مراجع الأسطر |
| 2 | وجّه بالأوضاع | `shift+tab`، plan، auto |
| 3 | تراجع عن أي شيء | `/rewind`، `Esc-Esc` |
| 4 | شغّل في الخلفية | المهام، `/tasks` |
| 5 | علّم Claude قواعدك | `CLAUDE.md`، `/memory` |
| 6 | وسّع بالأدوات | MCP، `/mcp` |
| 7 | أتمت سير عملك | skills، hooks |
| 8 | ضاعف نفسك | subagents، `/agents` |
| 9 | برمج من أي مكان | `/remote-control`، `/teleport` |
| 10 | اضبط النموذج | `/model`، `/effort` |
