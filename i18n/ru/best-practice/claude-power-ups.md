# Лучшие практики Power-ups

![Last Updated](https://img.shields.io/badge/Last_Updated-Apr%2002%2C%202026-white?style=flat&labelColor=555)

Интерактивные уроки, обучающие функциям Claude Code с анимированными демо. Каждый power-up обучает одной возможности Claude Code, которую большинство людей пропускает. Представлены в v2.1.90.

```bash
claude
/powerup
```

## Power-ups (10)

<p align="center">
  <img src="../../../best-practice/assets/claude-power-ups/powerup-menu.png" alt="Меню Power-ups с 10 уроками" width="700">
</p>

| # | Power-up | Темы |
|---|----------|------|
| 1 | Общайтесь с кодовой базой | `@` файлы, ссылки на строки |
| 2 | Управляйте режимами | `shift+tab`, plan, auto |
| 3 | Отмените что угодно | `/rewind`, `Esc-Esc` |
| 4 | Запуск в фоне | задачи, `/tasks` |
| 5 | Научите Claude своим правилам | `CLAUDE.md`, `/memory` |
| 6 | Расширяйте инструментами | MCP, `/mcp` |
| 7 | Автоматизируйте рабочий процесс | skills, hooks |
| 8 | Умножьте себя | subagents, `/agents` |
| 9 | Кодируйте откуда угодно | `/remote-control`, `/teleport` |
| 10 | Настройте модель | `/model`, `/effort` |
